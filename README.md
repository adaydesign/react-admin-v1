# Setup

- copy file .env.sample and rename to .env 
- change encrytion key at REACT_APP_ENCYPT_KEY

```
REACT_APP_ENCYPT_KEY=[your_encryption_key]
```

# Run
```
$ npm start
```

# Build

```
$ npm run build
```
